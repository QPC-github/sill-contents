#+title: Données du Socle Interministériel de Logiciels Libres

Cette page permet d'accéder aux fichiers ~json~ du Socle
Interministériel de Logiciels Libres (SILL).

Pour en savoir plus sur le SILL, lisez [[https://disic.github.io/sill/][cette page]].

Pour proposer un logiciel au SILL, ouvrez une issue sur [[https://github.com/disic/sill][ce dépôt]].

La liste des informations accessibles depuis cette page :

- [[file:sill.json][sill.json]] : les informations du SILL
- [[file:sill-contributors.json][sill-contributors.json]] : les organismes contributeurs
- [[file:sill-stats.json][sill-stats.json]] : quelques chiffres
